# C#调用OpenAI ChatGPT及Azure语音识别，语音输出
***
### 使用步骤
1. 注册OpenAI账号并获取相应的Key
2. 注册Azure服务获取语音服务的Key及地区信息
3. 在软件配置文件App.config中填入对应的信息（或者在软件中填写后点击应用）

### 注意
* ChatGPT使用[OpenAI-DotNet](https://github.com/RageAgainstThePixel/OpenAI-DotNet)库进行调用
* Azure使用[Microsoft.CognitiveServices.Speech](https://aka.ms/csspeech)库
* 界面使用[WPF-UI](https://github.com/lepoco/wpfui)样式
